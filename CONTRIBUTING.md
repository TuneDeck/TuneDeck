# Contributing

## Local development

```bash
npm install
npm run dev
```

## Before opening a PR

- make sure the app starts without Electron errors
- verify queue, search and genre pages manually
- verify the external API returns playable `audio_url` values

## Commit style

Keep commits short and focused. Example:

- `feat: add queue drawer`
- `fix: normalize artist arrays`
- `docs: expand readme`
